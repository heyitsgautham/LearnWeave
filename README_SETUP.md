## ğŸ¯ Quick Start Instructions

### Step 1: Run Automated Setup
```powershell
# From the Nexora root directory
.\setup.ps1
```

This will:
- âœ… Check all prerequisites
- âœ… Create .env file
- âœ… Generate secret keys
- âœ… Install Python dependencies
- âœ… Install Node.js dependencies

### Step 2: Get API Keys (30 minutes)

**REQUIRED:**
1. **Google Cloud Service Account** (15 min)
   - Go to https://console.cloud.google.com/
   - Create project â†’ Enable Vertex AI API
   - Create service account â†’ Download JSON
   - Save as `backend/google-credentials.json`

2. **MySQL Database** (10 min)
   - Run: `mysql -u root -p < backend/setup_database.sql`
   - Or create manually (see LOCALHOST_SETUP.md)

3. **Secret Keys** (1 min)
   - Already generated by setup.ps1!
   - Copy them from the output into your .env file

**OPTIONAL (but nice to have):**
4. **Unsplash API** (5 min) - https://unsplash.com/developers
5. **OAuth Providers** (5 min each) - Google/GitHub/Discord

### Step 3: Configure .env File
```powershell
# Edit backend/.env with your text editor
notepad backend\.env
```

Fill in at minimum:
```env
SECRET_KEY=<from setup.ps1 output>
SESSION_SECRET_KEY=<from setup.ps1 output>
DB_USER=nexora_user
DB_PASSWORD=<your mysql password>
GOOGLE_APPLICATION_CREDENTIALS=d:\Projects\Nexora\backend\google-credentials.json
```

### Step 4: Start ChromaDB
```powershell
cd backend\chroma_db
docker-compose up -d
```

### Step 5: Run the Application
```powershell
# Option A: Start everything at once
.\start.ps1 -All

# Option B: Start manually in separate terminals
# Terminal 1 - Backend
cd backend
.\venv\Scripts\activate
uvicorn src.main:app --reload

# Terminal 2 - Frontend  
cd frontend
npm run dev
```

### Step 6: Create Admin User
```powershell
cd backend
python create_admin.py
```

### Step 7: Open in Browser
ğŸ‰ Go to http://localhost:5173
